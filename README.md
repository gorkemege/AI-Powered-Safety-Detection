# Project Title: Construction Safety Detection with YOLOv8

Description: A computer vision project designed to detect Personal Protective Equipment (PPE) like hardhats and vests on construction sites.

Project Highlights:

Fine-tuned a YOLOv8 model on a custom dataset of 4,900+ images.

Achieved 98% mAP@50 accuracy during validation.

Integrated OpenCV for video stream processing and visualization.

Tech Stack: Python, YOLOv8, PyTorch, OpenCV.
